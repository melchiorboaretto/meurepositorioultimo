=== Exemplo 5: Persistência de Dados via Hardlinks ===

1. Criando arquivo original 'base.txt'...
2. Criando dois hardlinks para 'base.txt' ('hlink_1' e 'hlink_2')...
3. Deletando o arquivo original 'base.txt'...
4. Tentando ler dados através de 'hlink_1'...
   Leitura via 'hlink_1': Dados Persistentes
5. Deletando 'hlink_1'...
6. Tentando ler dados remanescentes através de 'hlink_2'...
   Leitura via 'hlink_2': Dados Persistentes
7. Deletando o último hardlink 'hlink_2' (Blocos devem ser liberados fisicamente do disco).

Teste concluído com sucesso.
